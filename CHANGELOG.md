# Changelog

## 1.0.9

- **PDF page orientation** (portrait / landscape) in the **Formats** panel.
- **PDF export CSS** textarea: optional styles injected only during export (good for dark themes).
- **Use page styles in PDF** (default on): reads computed colours, fonts, and table backgrounds from the live page plus export CSS.
- **PDF staging background** for exports from dark UIs.

## 1.0.8

Current release of **DataDump Export**. PDF export now snapshots live rendered HTML, applies striped table layout, and respects Bootstrap / `data-export-tone` cell styling. See [README](README.md).
